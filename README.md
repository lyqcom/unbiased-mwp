# Unbiased Math Word Problems Benchmark for Mitigating Solving Bias

This repository is the Mindspore implementation for the NAACL 2022 accepted paper


## Requirements
- python 3
- [PyTorch](http://pytorch.org/) 0.4.1


## Train and Test

- Math23K: 
```
python3 run_seq2tree.py
```
